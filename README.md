> **"Estejam sempre preparados para responder a qualquer que lhes pedir a razão da esperança que há em vocês." (1 Pe 3:15)**

O **Faith Forge** é uma plataforma web voltada para a apologética cristã (defesa racional da fé), focada no público protestante histórico. O sistema combina conteúdo teológico profundo com análise de dados, permitindo que usuários fortaleçam sua base doutrinária enquanto o sistema mapeia níveis de convicção e clareza através de uma dashboard analítica.

## 📖 Sobre o Projeto

Na era digital, o acesso à informação trouxe também uma avalanche de questionamentos céticos. O Faith Forge atua como uma "forja" intelectual, onde a dúvida é o fogo que purifica a fé através da lógica e das escrituras.

O diferencial do projeto é a **interatividade**: ao contrário de blogs estáticos, aqui o usuário avalia cada resposta, gerando métricas reais para líderes e educadores identificarem lacunas de conhecimento em suas comunidades.

## 🚀 Funcionalidades

- **Landing Page Informativa**: Apresentação da missão, metodologia e propósito do projeto.
- **Autenticação Segura**: Sistema de Cadastro e Login para acesso ao conteúdo exclusivo.
- **Biblioteca de Apologética**: Galeria interativa de perguntas e respostas sobre existência de Deus, ciência, confiabilidade bíblica e mais.
- **Sistema de Feedback**: O usuário avalia a clareza de cada explicação, alimentando a base de dados.
- **Dashboard Analítica**: Visualização em tempo real (via Chart.js) dos níveis de convicção e engajamento por tema.

## 📈 Metodologia de Desenvolvimento

Este é um Projeto Individual desenvolvido sob a metodologia Scrum (Adaptada):

- Product Backlog: Organizado no Trello com estimativas em Fibonacci.
- Sprints: Ciclos semanais de entrega de funcionalidades.
- Gestão: Foco em autogestão e refatoração constante de código.

## 🏗️ Como Executar o Projeto

### Requisitos para a instalação:
![NodeJS](https://img.shields.io/badge/node.js-6DA55F?style=for-the-badge&logo=node.js&logoColor=white)

### Configuração do Banco de Dados:
Execute o script SQL fornecido no repositório para a inicialização do banco de dados.

### Instalação e Execução:

1. Instale as dependências do projeto:
```bash
npm i
```

2. Inicie o servidor após a instalação de todas as dependências:
```bash
npm start
```
